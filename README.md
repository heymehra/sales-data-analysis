# Sales Data Analysis

This project performs exploratory data analysis (EDA) on sales data using Python.
The analysis focuses on identifying sales trends, daily patterns, spikes, and product-level insights
using pandas and matplotlib.

## Dataset
- `sales_data.csv`: Contains order-level sales data including date, product, quantity, unit price, and total sales.

## Key Analysis
- Sales trends over time
- Day-wise sales performance
- Product-level quantity and revenue analysis
- Identification of sales spikes and variability

## Tools Used
- Python
- pandas
- matplotlib
